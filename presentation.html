Platform Overview

Core Purpose

  • Centralized management for high-performance petrotechnical workstations.
  • Comprehensive infrastructure monitoring and reporting.
  • Automated IT helpdesk and licensing workflows.

Key Features

  • Dual-mode VDI Access (Real RDP + Legacy Sim).
  • Real-time VM & Storage Telemetry.
  • Enterprise License Tracking & Auditing.
  • Role-based access (Spatie Permissions).

Modular design with strict 1-way data flow: Controller → Service → Model

VDI Module: Secure Desktop Access

Dual-Mode Connectivity

  • Real RDP: Integrated Apache Guacamole for browser-based remote desktop access.
  • Legacy Sim: Fullscreen web-based OS simulation for training/demo environments.

Security & Integration

  • RDP credentials stored with AES-256 encryption.
  • Automatic session termination & Guacamole connection cleanup.
  • Connection tokens for session lifecycle management.

Infrastructure Monitoring

VM Metrics

  • 24-hour time-series telemetry for CPU, RAM, Disk, & Network.
  • GPU Utilization tracking with specialized hardware support.
  • Dynamic trend charts powered by ApexCharts.

Storage Insights

  • Capacity monitoring for NAS/SAN/Object Storage.
  • 30-day historical usage projection & growth analysis.
  • Real-time status tracking across multiple regions.

License & Analytics

Software Compliance

  • Centralized FlexLM & custom license server tracking.
  • Automated expiry warnings and seat usage auditing.
  • Spatie Activity Logs for every change (Audit Trail).

Platform Analytics

  • Aggregated dashboards for system-wide health.
  • Time-period filtering (Week / Month / Year).
  • Radial & Bar charts for resource allocation overview.

Support & Ticketing

Helpdesk Workflow

  • Seamless user reporting for VDI or License issues.
  • Internal notes & thread management for IT admins.
  • Automated ticket numbering based on date patterns.

RBAC Policies

  • Hierarchical roles: Super Admin, Admin, User.
  • Granular permission policies for every model interaction.
  • Privacy controls via internal/external comment flags.

Tech Stack & Workflow

Modern Stack

  • Laravel 12 (Modern PHP 8.2+)
  • PostgreSQL 18 (Performance)
  • Docker (Isolation)
  • ApexCharts (Visuals)

Workflow Protocol

  • Strict Plan-Implement-Verify-Document cycle.
  • No business logic in Models (Service Layer Pattern).
  • Full route & view caching for production performance.
